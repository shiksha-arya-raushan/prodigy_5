Weather Dashboard - Web Development Task
A modern, responsive Weather Dashboard that provides real-time weather updates for any city in the world. This project was developed to demonstrate proficiency in handling asynchronous API calls and creating a seamless user experience.
🚀 Key Features
Live Weather Updates: Fetched directly from the OpenWeatherMap API.
Detailed Metrics: Displays temperature, humidity, wind speed, and weather conditions.
Interactive Search: Allows users to search for any city globally.
Mobile First: Designed with a responsive layout to work perfectly on smartphones (developed using Acode and Spck Editor).
🛠️ Technical Stack
HTML5: Semantic structure for the dashboard components.
CSS3: Custom styling including Flexbox/Grid for responsiveness and modern UI aesthetics.
JavaScript (ES6): Used fetch() for API integration and dynamic DOM manipulation.
Version Control: Managed and deployed via GitHub.
📂 Project Structure
weather.html: The core structure of the application.
style4.css: Custom styles for the weather cards and backgrounds.
script3.js: Logic for API fetching and data rendering.
